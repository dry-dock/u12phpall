u12phpall
===============
Shippable CI image for php on Ubuntu 12.04. Available php versions: 
 1. php5.4
 2. php5.5
 3. php5.6
 4. php7.0
 
## Services:

1. cassandra 2.1
2. elasticsearch 1.5
3. memcached 1.4
4. mongodb 3.0
5. mysql 5.6
6. neo4j 2.2
7. postgres 9.4
8. rabbitmq 3.5
9. redis 3.0
10. selenium 2.45
11. sqllite 3
12. couchdb 1
